About This Project

This website is a live demonstration of Keystone—a modular, cost-optimized AWS environment managed via Terraform. It features a containerized Flask application on ECS Fargate , secured with Cloudflare Tunnels for ingress, Github Packages for container registry and integrated with Grafana Cloud for advanced observability, all automated through a GitHub Actions CI/CD pipeline.
